# Natij Portfolio - ناتج

Personal portfolio website for Yahia Mohammed (Natij).

## Tech Stack

- HTML5 (Semantic)
- CSS3 (Custom Properties + Animations)
- JavaScript (Vanilla ES6+)
- GSAP 3.15.0 (CDN)
- Hosting: GitHub Pages

## Features

- Responsive design
- GSAP animations
- Anti-theft protection (watermark, disable right-click)
- Portfolio filter
- Contact form (mailto)
- Mobile-friendly

## Setup

1. Clone the repository
2. Replace `assets/logo.png` with your logo
3. Add portfolio items to `index.html`
4. Deploy to GitHub Pages

## Security

- Right-click disabled
- Keyboard shortcuts blocked
- Image drag disabled
- Watermark overlay active
